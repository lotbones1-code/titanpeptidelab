Lab testing

Every lot documented, one release standard.

Each lot is released against an in-house analytical release sheet. Checks Titan does not run in-house are described as outside-lab tested only when a named report is available for that lot. If a documented release check falls outside its target, the batch is held. The point is not to print an impressive certificate — it is to make the certificate believable.
  1. 01

    Identity

    Method
    Mass-spectrometry identity check
    Spec
    Observed mass recorded against the theoretical [M+H] for the labeled compound

    Confirms the compound in the bottle is the compound on the label. Purity of the wrong molecule is still a failed batch.

  2. 02

    Purity

    Method
    HPLC-UV purity check
    Spec
    Release target ≥ 99% main-peak area; lots below target are held

    The number most buyers look for first. The release value is recorded on the lot-release sheet referenced to your bottle's lot code.

  3. 03

    Sterility

    Method
    Sterility check (USP <71>-style)
    Spec
    Target: no growth over the documented incubation period

    Applied to nasal sprays and reconstituted liquids. The read is recorded on the lot-release sheet before release is signed.

  4. 04

    Endotoxin

    Method
    Endotoxin (LAL) check
    Spec
    Reference limit < 5 EU/mg

    A clean chromatogram on a pyrogenic lot is still a failed batch. This filter separates a real QA operation from a cosmetic one.

  5. 05

    Heavy metals

    Method
    ICP-MS (Pb, Cd, As, Hg)
    Spec
    Within ICH Q3D Option 1 limits

    Not run in-house. Heavy-metals results are shown only when a named outside-lab report is available for the lot — Titan does not claim instrumentation it does not operate.

  6. 06

    Residual solvents

    Method
    Residual-solvent screen
    Spec
    Reference: ICH Q3C class 2 solvents at report threshold

    DMF, methanol, acetonitrile, and related residue checks are screened and recorded on the retained lot documentation.

Batch tracking.

Every bottle leaves with a lot number of the form TPL-YYMM-[A-Z]. That same lot appears on the bottle, the order record, and the certificate. If you lost the printed copy, email support@titanpeptidelab.com with the lot number and a signed PDF can be reissued.
