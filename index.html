Quality system

The certificate stays attached to the bottle, not just the brand.

The whole point of the workflow is keeping batch identity intact. Compound, certificate, QR lookup, and shipping note all resolve to the same lot instead of a generic promise page.

Certificate lookup path

  1. 01

    Lot intake

    Every nasal spray batch gets a Titan lot code before fill. That code stays tied to the bottle, internal record, certificate, and checkout reference.

  2. 02

    HPLC purity screen

    Release threshold is HPLC purity of at least 99%. Chromatogram, method, test date, and reviewer initials stay attached to each lot record.

  3. 03

    COA match

    The certificate references the exact same lot carried through the order workflow. No representative COAs get swapped across batches.

  4. 04

    Cold-chain dispatch

    Liquid orders are packed for temperature control, fulfilled manually, and sent with tracking plus the matching certificate within 24 hours.
